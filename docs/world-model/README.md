## 关键论点与论文

- World Models (Ha & Schmidhuber 2018) —— 概念奠基
- Sora technical report (2024) —— "world simulator" 主张
- Genie (DeepMind 2024) —— 可交互、动作可控
- V-JEPA (Meta 2024) —— 预测式表征学习路线（与生成式对立的一条路）
